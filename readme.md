<h2> APIs </h2>
  <h3>/list</h3>
  returns all articles
  <h3>/filter</h3>
  returns all articles matching the filter<br>
  <h5>example:</h5>
  <code>filter?TOPICS=grain,wheat,barley&DATE=APR</code><br>
  <h5>Applicable filtering tags:</h5>
    DATE,<br>
    UNKNOWN,<br>
    TOPICS,<br>
    PLACES,<br>
    PEOPLE,<br>
    ORGS,<br>
    EXCHANGES,<br>
    COMPANIES,<br>
    TEXT,<br>
    TITLE,<br>
    DATELINE,<br>
    AUTHOR,<br>
    BODY;<br>
   
  <h3>/id</h3>
  returns an article with matching newReuters id
  <h5>example:</h5>
  <code>id/17022</code>
